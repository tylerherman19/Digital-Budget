# Digital Budget Tracker

A single-page dashboard for tracking digital media spend against budget across channels (Google, Meta, DSP), with pacing analysis and weekly burn visualizations.

**Live site:** https://tylerherman19.github.io/Digital-Budget/

## Tabs

- **Summary** — hero pacing card and high-level KPIs.
- **Insights & Charts** — cumulative burn vs target pace and spend mix by week.
- **Pacing & Analysis** — cumulative and weekly spend vs plan, with channel drill-down.

## Running locally

The app is a single `index.html` file with no build step. Open it directly, or serve the directory with any static file server:

```sh
python3 -m http.server 8000
# then visit http://localhost:8000
```
